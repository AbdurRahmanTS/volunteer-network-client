# Volunteer Network

Admin can add a Volunteer service event.<br />
Users can register for volunteer service.<br />
Admin can delete any registered event.

**Technology:** <br />
React.js<br />
Node.js<br />
Heroku<br />
Firebase<br />
MongoDB<br />
React Router<br />
HTML5<br />
CSS3<br />
Material UI.
